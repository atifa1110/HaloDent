# HaloDent

### Overview :
Halo Dent adalah Aplikasi Messanger Koas Mahasiswa dengan pasien yang diawasi oleh dokter pengawas. Dimana aplikasi ini melibatkan 3 user yaitu konselor, dokter, dan pasien.

### Features :
1. Menampilkan Konselor yang sedang Online
2. Menampilkan Jadwal Konsultasi
3. Menampilkan Detail Konselor
4. Menampilkan Detail Dokter Pengawas
5. Menampilkan List Berita terkini
6. Melakukan Kuesioner Karie gigi
7. Melakukan Login/Logout
8. Melakukan Edit Profile
9. Melakukan Group Chat : Pertukaran pesan dan gambar
10. Melakukan Group Info
11. Menampilkan Notifikasi
